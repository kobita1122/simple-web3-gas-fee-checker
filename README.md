# Simple Web3 Gas Fee Checker

This is a beginner-friendly blockchain project.

This project helps you:
- Check current gas fees
- See low, medium, and high fee levels
- Understand how transaction fees work

This project uses:
- HTML
- JavaScript
- Public blockchain API

## Features
- One-click gas fee check
- Show low, medium, high fees
- Simple layout
- Easy to edit code

## Requirements
- Internet connection
- Basic JavaScript knowledge

## How to Run
1. Download this repository
2. Open `index.html` in browser
3. Click "Check Gas Fees"
4. See results on screen

## File Structure
- index.html -> Main webpage
- app.js -> App logic
- config.js -> API settings
- api.js -> Gas API calls
- low.js -> Low gas logic
- medium.js -> Medium gas logic
- high.js -> High gas logic
- ui.js -> Screen updates
- format.js -> Format numbers
- utils.js -> Helper functions

## Learning Goal
This project helps beginners learn:
- What gas fees are
- How blockchain pricing works
- How APIs return live data
